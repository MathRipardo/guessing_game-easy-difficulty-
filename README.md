# 🎯 Guessing Game (Python)

A simple number guessing game made with Python.
This project was created as a beginner exercise while studying Software Engineering.

## 📌 About the Project

In this game, the computer randomly chooses a number and the player must guess it.
The program gives hints if the number is higher or lower until the player finds the correct answer.

## 🚀 Features

* Random number generation
* User input
* Hints (higher or lower)
* Unlimited attempts until the correct guess

## 🛠 Technologies Used

* Python
* Random module

## ▶️ How to Run

1. Clone this repository
2. Open the project folder
3. Run the file:

```
python guessing_game.py
```
